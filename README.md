# VillageDefender



## Contrôles Caméra

| Touche(s)               | Action |
|-------------------------|--------|
| Z / S / Q / D           | Déplacer la caméra |
| Q / E                   | Rotation gauche / droite |
| Clic molette (maintenu) | Rotation libre (haut/bas et gauche/droite) |
| Molette                 | Zoom avant / arrière |
| T                       | Basculer vue du dessus |

> La caméra se déplace également en approchant la souris des bords de l'écran (edge scrolling).

## Contrôles Menu

| Touche(s) | Action |
|-----------|--------|
| P         | Ouverture du menu |
